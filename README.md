# Docker Learning 🐳

This repository contains my Docker learning projects while exploring containerization.

## Project 1 - Flask Multiplication Table

A simple Flask web application that generates the multiplication table of a number entered by the user.

### Technologies

- Python
- Flask
- Docker

### Docker Commands Practiced

```bash
docker build -t gunjanahujaa/flask-table .
docker run -p 5000:5000 gunjanahujaa/flask-table
docker image ls
docker container ls
docker ps
```
